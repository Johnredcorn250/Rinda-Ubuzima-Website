# Rinda Ubuzima Website

A health-focused website project.

## Project Structure

```
Rinda-Ubuzima-Website/
├── index.html        # Main HTML entry point
├── css/
│   └── style.css     # Global styles & responsive layout
├── js/
│   └── main.js       # JavaScript (nav toggle, smooth scroll)
└── assets/           # Images, fonts, icons (add as needed)
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Johnredcorn250/Rinda-Ubuzima-Website.git
   ```
2. Open `index.html` in your browser to preview locally.
3. Use a code editor (e.g., VS Code) to make changes.

## Tech Stack

- HTML5
- CSS3 (custom, responsive)
- Vanilla JavaScript
